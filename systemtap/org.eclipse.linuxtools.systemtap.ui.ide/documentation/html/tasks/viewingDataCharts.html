Viewing Data Charts

Viewing data charts from a SystemTap script is accomplished by running that script with the Run->Run w/ Chart option. This is covered in the IDE Tutorial: Lesson Three. After running a script with the chart option, the user is taken into the Graphing Perspective and the Data Table is loaded by default. The data table immediately begins populating as it recieves input back through the console from the running SystemTap script.
