Graph Interaction

The graphical options used in the Dashboard perspective:
  • Zoom Bar - This enables the user to zoom in and out on the current graph by sliding the bar to the left and right. The current magnification is shown at the top of the bar. Zooming would increase the no of x-axis ticks therefore allowing the user to see more(or less) data at a point in time.

  • - Show Zoom bar - This displays the zoom bar.
  • - Close Zoom bar - This hides the zoom bar.
  • - Close Graph - This removes the graph from the viewer pane.
